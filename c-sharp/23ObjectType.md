# Object Type

The type from which all dotnet objects inherit. It constains a few default methods that can be very useful. You can change the type of the value contained in an object, you can have lists of objects with elements of different types because object type is the tyope of every object.

## Methods

- Equals: Used in lists and dictionaries in order to do comparisons.
- GetHashCode: Gives the hash of our object. It is used in dictionaries in order to access an object.
- GetType: get the type of the object underneath. If you put a string in an object, it will return string as the gettype.
- ToString: Shows a string representation of our object.
