# Status codes

200 - Success
201 - created
202 - Accepted

300 - Multiple choices
301 - moved permanently. You get a new url
302 - temporary change
303 - Redirection

400 - bad request
401 - Unauthorized ( unauthenticated )
403 - Forbidden (we know you and it's a no)
404 - Not found
408 - Timeout

500 - Internal Server Error
501 - Not implemented
502 - Bad gateway
503 - Unavailable
504 - Gateway Timeout
